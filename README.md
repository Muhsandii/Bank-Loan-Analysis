# 💳 Bank Loan Analysis Dashboard
Data Source: https://www.kaggle.com/datasets/datawitharyan/financial-loan-dataset/data

An end-to-end Data Analytics project that analyzes loan portfolio performance and credit risk using **PostgreSQL** and **Power BI**.

## 📌 Project Overview

This project analyzes historical loan data to evaluate portfolio performance, identify high-risk borrowers, and provide business insights that support better lending decisions.

## 🛠️ Tools

- PostgreSQL
- SQL
- Power BI
- Git & GitHub

## 📈 Key Insights

- Good Loans account for approximately **86%** of the portfolio.
- Grade **G** has the highest default rate.
- **Small Business** loans present the highest credit risk.
- Borrowers with higher **DTI** tend to have higher default rates.
